# Machine Learning Labs

Labs completed as part of the **Machine Learning** course at the Faculty of Computer Science and Engineering — Skopje (FINKI), 2025/2026.

Each lab includes the task description (PDF) and a Jupyter Notebook with the full implementation.

---

## Labs

### Lab 01 — Classification

**Dataset:** MAGIC Gamma Telescope

Binary classification task — distinguishing gamma-ray signals from hadron background using telescope observation data.

**Topics:** Supervised learning, classification algorithms, model evaluation

📓 [236006_lab01.ipynb](lab01-classification/236006_lab01.ipynb) · 📄 [task.pdf](lab01-classification/task.pdf)

---

### Lab 02 — Regression

**Dataset:** Diamond Price Prediction

Predicting diamond prices based on physical and quality attributes (carat, cut, color, clarity, dimensions).

**Topics:** Regression models, feature engineering, evaluation metrics (RMSE, R²)

📓 [236006_lab02.ipynb](lab02-regression/236006_lab02.ipynb) · 📄 [task.pdf](lab02-regression/task.pdf)

---

### Lab 03 — Unsupervised Learning

**Dataset:** Mice Protein Expression

Clustering 77 protein expression features without class labels to discover natural groupings that reflect true classes.

**Topics:** K-Means, DBSCAN, dimensionality reduction, cluster evaluation

📓 [236006_lab03.ipynb](lab03-clustering/236006_lab03.ipynb) · 📄 [task.pdf](lab03-clustering/task.pdf)
